# Salary Prediction Using Linear Regression

A Machine Learning project that predicts employee salaries based on years of experience using Linear Regression.

## Features

- Salary prediction
- Data visualization
- Linear regression model
- Residual analysis
- Model evaluation using RMSE and R2 score

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Statsmodels

## Dataset

The project uses a salary dataset containing:
- YearsExperience
- Salary
